== Description ==

This WordPress plugin allows you to easily create tweetable content for your readers. Use pullquotes and custom messages inside the body of your blog post to highlight your content for the reader and make it simple for them to share on Twitter.

This is a plugin by CoSchedule, the [all-in-one content marketing editorial calendar for WordPress](http://coschedule.com).

== Installation ==

**To install the plugin manually in WordPress:**

1. Login as Admin on your WordPress blog.
2. Click on the "Plugins" tab in the left menu.
3. Select "Add New."
4. Click on "Upload" at the top of the page.
5. Select the 'tm-click-to-tweet.zip' on your computer, and upload. Activate the plugin once it is uploaded.

**To install the plugin manually with FTP:**

1. Unzip the 'tm-click-to-tweet.zip' file. Upload that folder to the '/wp-content/plugins/' directory.
2. Login to your WordPress dashboard and activate the plugin through the "Plugins" tab in the left menu.
