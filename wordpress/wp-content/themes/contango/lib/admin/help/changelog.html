Contango WordPress Theme Changelog Information

1.2.4 [2013.12.10]

Maintenance Release

  1. Breadcrumbs support has been added.
  2. Single Post and Pages are updated to support the breadcrumbs.
  3. Supported Plugins
    • WordPress SEO by Yoast: http://wordpress.org/plugins/wordpress-seo/
    • Breadcrumb NavXT: http://wordpress.org/plugins/breadcrumb-navxt/

1.2.3 [2013.12.05]

Maintenance Release

  1. Custom Header logic has been updated.
  2. Flexible width for custom header has been added to faciliate the header of any size.
  3. Some CSS changes to control the display of flexible header.

1.2.2 [2013.11.24]

Maintenance Release

  1. HTML Markup has been updated to validate Google Authorship Structured Data. i.e. hatom-feed
  2. Microformats "time" element has been added.
  3. Support for popular SEO plugins has been added. i.e. All In One SEO Pack, WordPress SEO

1.2.1 [2013.11.18]

Maintenance Release

  1. Post Format support has been added.

1.2 [2013.09.21]

Maintenance Release

  1. Theme Options logic has been updated.

1.1.1 [2013.06.26]

Maintenance Release

  1. Code maintenance to avoid white screen on some hosting accounts while uploading custom header as reported by users.
  2. Data sanitation has been improved.
  3. Theme Options panel has been updated.

1.1 [2013.06.02]

Maintenance Release

  1. Superfish supports default menu drop down when there is no menu selected at "Primary Menu" theme location.
  2. Primary Menu "depth" has been updated to five levels.
  3. Layout of Older/Newer comments has been fixed.
  4. Non Breaking text has been fixed in the content.
  5. Theme Options panel has been updated.

1.0 [2013.03.05]

Initial Release.
