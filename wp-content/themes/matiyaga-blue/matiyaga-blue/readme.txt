Thank you for downloading Matiyaga Blue WordPress theme. Be sure you have downloaded the latest version of this theme at http://www.paddsolutions.com/wordpress-theme-matiyaga

Installation
============
1. Extract matiyaga-blue.zip and upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress Admin Dashboard > Appearance > Themes and select the Matiyaga Blue screenshot.
3. Activate Matiyaga WordPress Theme.

Required plugins and scripts
============================
1. Upload the tweetmeme and wp-pagenavi folders in the /wp-content/plugins directory.
2. Activate TweetMeme and WP-PageNavi plugins.
